Java基础之基本数据类型

Author: ChenJunYu Date: February 9, 2021  14:15:00 Category: Java基础

Java八大基本数据类型

类型 所占字节 默认值 取值范围 包装类
byte(字节型) 1 0 -128～127(-2⁷~2⁷-1) Byte
short(短整型) 2 0 -32768～32767(-2¹⁵~2¹⁵-1) Short
char(字符型) 2 \u0000(空格) Unicode字符 Character
float(浮点型) 4 0.0f 3.4e-45~1.4e+38 Float
int(整形) 4 0 -2147483648～2147483647(-2³¹~2³¹-1) Integer
double(双精度浮点型) 8 0 4.9e-324~1.8e+308 Double
long(长整型) 8 0.0l -9223372036854774808～9223372036854774807(-2⁶³~2⁶³-1) Long
boolean(布尔型) / false true/false Boolean

类型转换

顾名思义，将一种类型转换成另一种类型。除了boolean类型的值无法，其他七种基本类型都可以互相转换，但是可能存在精度丢失或其他一些特殊情况。类型分为自动转换和强制转换，自动转换无需任何操作，而强制转换需要显式转换。

byte <（short=char）< int < long < float < double

相对于取值范围进行排序，如果从小转换到大，可以自动完成，而从大到小，必须强制转换。short和char两种相同类型也必须强制转换。

自动转换

public class Test {
    public static void main(String[] args) {
        int a = 10;
        double b = 10d;
        System.out.println(a+b);
    }
}

运行

由此可见，类型已经被转换成了double。
如果类型为char型，向整型转换时，会转换为对应ASCII码值（ASCII（American Standard Code for Information Interchange，美国信息交换标准代码）是基于拉丁字母的一套电脑编码系统。

public class Test {
    public static void main(String[] args) {
        char c = 'A';
        int a = c;
        System.out.println(a);
    }
}

运行

强制转换

将”大”数据转换为”小”数据时，你可以使用强制类型转换。这种转换肯定可能会导致溢出或精度的下降。

public class Test {
    public static void main(String[] args) {
        int a = 520;
        System.out.println((byte) a);
    }
}

结果

这是因为a的值为520，超出了byte的范围(-128~127)，所以将520除以byte的范围（256）取余数得到8。记住，将浮点类型赋给整数类型的时候，会发生截尾，也就是把小数的部分去掉，只留下整数部分。
