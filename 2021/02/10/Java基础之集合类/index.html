Java基础之集合类

Author: ChenJunYu Date: February 10, 2021  16:13:00 Category: Java基础

Set和List的区别

List,Set都是继承自Collection接口，都是用来存储一组相同类型的元素的。
Set 无序，不可重复
List 有序，可重复

ArrayList和LinkedList和Vector的区别

这三者都实现了List接口。
ArrayList是一个可改变大小的数组，当更多的元素加入到ArrayList中时,其大小将会动态地增长，ArrayList本质上就是个数组。
LinkedList是一个双链表，与ArrayList的区别就是在增加或者删除时效率会比ArrayList快，但在get和set时就弱于ArrayList。
Vector和ArrayList类似，但Vector是线程安全的。
Vector和ArrayList在更多元素添加进来时会请求更大的空间。Vector每次请求其大小的双倍空间，而ArrayList每次对size增长50%.

Set是如何保证元素不重复的？

在Set体系中，有两大类：TreeSet和HashSet，两者有什么区别呢

TreeSet是二叉树实现的，数据自动排序，不允许插入null值。
HashSet是哈希表实现的，数据无序，可以插入null值。
HashSet的底层是HashMap，当向HashSet添加数据的时候，首先会计算元素的hashcode值，然后通过扰动计算和按位与的方式计算这个元素的存储位置，如果这个位置为空，则插入，否则就用equals判断元素是否相等，相等则不添加，否则找一个空位置插入。
TreeSet的底层是TreeMap的keySet()，而TreeMap是红黑树实现的，红黑树是一个平衡二叉查找树，它能保证任何一个节点的左右子树高度差不会超过较矮的那棵的一倍。
TreeMap是按key排序的，元素在插入TreeSet时compareTo()方法要被调用，所以TreeSet中的元素要实现Comparable接口。TreeSet是用compareTo()来判断重复元素的。
